# A Modular Cluster for AI Research in Resource-Constrained Academic Environments

**Author:** Bc. Filip Sršeň

**Supervisor:** Mgr. Ján Kľuka, PhD.

**Institution:** Comenius University in Bratislava, Faculty of Mathematics, Physics and Informatics

**Program:** Applied Informatics (2026)

## Short Description

This master thesis explores the design and implementation of a modular computing cluster tailored for AI research in academic environments with limited resources. The project aims to provide a cost-effective, scalable infrastructure that enables researchers and students to conduct machine learning experiments without relying on expensive commercial cloud services.

Key aspects of the project include:

- **Modularity** — the cluster can be incrementally expanded with additional compute nodes
- **Resource efficiency** — optimized scheduling and sharing of GPUs and other accelerators
- **Accessibility** — simple interfaces for researchers to deploy and manage ML workloads
- **Reproducibility** — infrastructure-as-code approach to ensure consistent environments

The thesis covers the full pipeline from architectural design and software implementation to evaluation of the cluster's performance on representative AI workloads.
